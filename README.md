# Календарь дня

Веб-приложение на JavaScript для отображения текущей даты и ведения заметок на выбранный день.  
Данные хранятся локально в браузере с использованием `localStorage`.

## Возможности

- Отображение текущей даты (день недели, число, месяц, год).
- Выбор любой даты через `<input type="date">`.
- Создание и редактирование заметки для выбранного дня.
- Автоматическое сохранение заметок в `localStorage`.
- Удаление заметки, если поле очищено.
- Статус-строка с информацией о загрузке и сохранении.

## Технологии

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## Запуск проекта

1. Склонировать репозиторий:

   ```bash
   git clone https://github.com/ВАШ_ЛОГИН/day-calendar.git
   ```

2. Перейти в папку проекта:

   ```bash
   cd day-calendar
   ```

3. Открыть файл `index.html` в браузере.
